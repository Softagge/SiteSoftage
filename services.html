Código Limpo e Moderno

Todos os nossos códigos fontes são escritos de forma limpa e moderna, tornando-o assim de fácil manutenção, proporcionando maior rapidez ao processo de manuntação dos códigos.

Qualidade de Software

Desenvolvemos projetos com processos de qualidade de software bem definidos, proporcionando aos nossos clientes um produto final que satisfaça suas expectativas, dentro daquilo foi acordado inicalmente.

Sistemas Adaptáveis

Trabalhamos com o intuito de propocionarmos aos nossos clientes sistemas responsivos e que são flexíveis a mudanças, podendo assim se adaptar-se as necessidades de um ambiente coorporativo.

A SOFTAGE

Missão

Nossa missão não se limita a sermos bons em qualidade, mas sim atingirmos níveis de excelência em qualidade. Para conquistar nossos clientes a longo prazo precisamos buscar continuamente a liderança em qualidade, inovação tecnológica, confiabilidade e preços competitivos, bem como assegurar prazo de entrega e assistência técnica.

Visão

Nossa visão é ser uma empresa de referência, reconhecida como a melhor opção por clientes, colaboradores, comunidade, fornecedores e investidores, pela qualidade de nossos sistemas, serviços e relacionamento.

Valores

Nossos valores:
Ética Inquestionável
Compromisso com o sucesso de nossos clientes
Flexibilidade e agilidade com um propósito
Espírito de equipe
Respeito à individualidade e à diversidade
Responsabilidade

Nossos Recursos

Padrões de Projeto

Fazemos o uso de alguns padrões de projeto como, MVC - Model View Controller , que é muito útil para resolver problemas de modelagem de projetos.

Metodologia Ágeis

Fazemos o uso da metodologia ágil XP - Extreme Programming, que facilita o desenvolvimento e utiliza de algumas praticas como "Programação em Par".

Plataforma Web

Desenvolvemos sistemas voltados a plataforma Web com o objetivo de facilitar o acesso, diminuir os itens de instalação e ter uma maior flexibilidade nas tarefas.

FrameWorks

Fazemos o uso de algumas frameworks como Java e PHP que facilita o desenvolvimento e que dita o fluxo de controle da aplicação, chamado de Inversão de Controle.

Banco de Dados

Fazemos o uso do MySQL - Meu Structured Query Language que é um sistema de gerenciamento de banco de dados(SGBD).
